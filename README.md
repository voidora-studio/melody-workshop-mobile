## 说明

**旋律工坊** 是基于开源项目 [LX Music（洛雪音乐助手）移动版](https://github.com/lyswhut/lx-music-mobile) 进行二次开发、品牌重设计与功能增强的派生版本。

所用技术栈：

- React Native 0.73
- TypeScript

已支持的平台：

- Android 5 及以上

***注：目前没有计划支持 iOS 和 HarmonyOS NEXT。**<br>
桌面版项目地址：<https://github.com/voidora-studio/melody-workshop>*

### 与本版相关

- **品牌**：界面、图标、主题色已按「旋律工坊」统一设计
- **署名**：本发行版由 **voidora-studio** 维护与分发

### 上游说明

本软件基于上游开源项目衍生而来，遵循其 Apache-2.0 许可。如需获取原版功能、提交 Issue 或参与上游开发，请访问：
- 上游仓库：https://github.com/lyswhut/lx-music-mobile
- 上游发布：https://github.com/lyswhut/lx-music-mobile/releases
- 使用问题可先参阅上游整理的常见问题：https://lyswhut.github.io/lx-music-doc/mobile/faq

### 数据同步服务

上游提供了独立的[数据同步服务](https://github.com/lyswhut/lx-music-sync-server#readme)。如果有服务器，可部署为私人多端同步服务。

### 开放 API 支持

从上游 v2.7.0 起支持开放 API 服务。启用后将在本地启动 HTTP 服务，供第三方软件调用播放器功能。

## 功能特色

- 多平台音乐源搜索与播放（酷我、酷狗、咪咕等）
- 歌单管理、自定义列表
- 歌曲下载（带 LRC 歌词文件）
- 桌面歌词悬浮窗（支持多行、锁定、透明度调节）
- 数据同步（跨设备同步播放列表）
- 主题编辑器（自定义主题色、背景图片、旋律工坊主题包）
- 听歌统计与记录
- 音乐评论浏览
- 逐字歌词进度显示
- 多列表管理

## 贡献代码

本项目欢迎 PR，但为了 PR 能顺利合并，需要注意以下几点：

- 对于添加新功能的 PR，建议在提交 PR 前先创建 Issue 进行说明，以确认该功能是否确实需要。
- 对于修复 bug 的 PR，请提供修复前后的说明及重现方式。
- 对于其他类型的 PR，则适当附上说明。

贡献代码步骤：

1. 参照上游[源码使用方法](https://lyswhut.github.io/lx-music-doc/mobile/use-source-code)设置开发环境；
2. 克隆本仓库代码并切换至 `dev` 分支进行开发；
3. 提交 PR 至 `dev` 分支。
